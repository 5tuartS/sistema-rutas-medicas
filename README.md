# sistema-rutas-medicas




Funcionalidad: Buscar si existe una ruta entre dos centros
Esta funcionalidad permite verificar si existe una ruta directa o indirecta entre dos centros médicos dentro del grafo. Se implementa usando el algoritmo de recorrido en anchura (BFS).

¿Qué hace?
Recorre el grafo desde el centro de origen.

Verifica si se puede llegar al centro de destino.

Devuelve true si hay conexión, false si no.
